# Настройка AI OCR для импорта PDF

Для работы AI OCR (распознавания текста через Google Gemini) необходимо настроить API ключ.

## 1. Получение API ключа Gemini

1. Перейдите на [Google AI Studio](https://aistudio.google.com/app/apikey)
2. Войдите в аккаунт Google
3. Нажмите "Create API Key"
4. Скопируйте ключ (он начинается с `AIza...`)

## 2. Настройка API ключа

### Способ 1: Через настройки приложения (рекомендуется для мобильных)

1. Откройте приложение
2. Перейдите в **Настройки**
3. Найдите пункт **"Gemini API ключ"**
4. Нажмите на него
5. Введите ваш API ключ
6. Нажмите **"Сохранить"**

### Способ 2: Переменная окружения (для разработки)

**Windows (PowerShell):**
```powershell
$env:GEMINI_API_KEY="AIza...твой_ключ_здесь"
flutter run
```

**Windows (CMD):**
```cmd
set GEMINI_API_KEY=AIza...твой_ключ_здесь
flutter run
```

**Linux/macOS:**
```bash
export GEMINI_API_KEY="AIza...твой_ключ_здесь"
flutter run
```

## 3. Как это работает

1. PDF файл разбивается на страницы через `pdfrx`
2. Каждая страница рендерится в изображение (PNG, 300 DPI)
3. Изображение конвертируется в base64
4. Отправляется запрос в Google Gemini API
5. AI распознает текст с изображения
6. Распознанный текст парсится через `KaspiStatementParser`
7. Транзакции сохраняются в базу данных

## 4. Стоимость

Google Gemini API:
- **Gemini 2.0 Flash**: Бесплатно до 15 запросов в минуту
- Для одной страницы PDF: ~1 запрос
- Для выписки из 10 страниц: ~10 запросов

**Рекомендация:** Gemini API имеет бесплатный лимит, что делает его более доступным, чем OpenAI.

## 5. Альтернативные методы

Если AI OCR недоступен или не настроен, приложение автоматически попробует:
1. ML Kit OCR (Google ML Kit) - офлайн распознавание
2. Поиск паттернов в бинарных данных
3. Декомпрессия PDF потоков

## 6. Безопасность

⚠️ **НИКОГДА не коммитьте API ключи в репозиторий!**

- Для мобильных приложений используйте настройки в приложении
- Ключ хранится локально в SharedPreferences
- Для CI/CD используйте секреты (GitHub Secrets, GitLab CI Variables)
